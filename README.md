# API REST con Node.js y Express

Una API REST simple para usuarios con persistencia en archivo `usuarios.json`.

## Rutas

- GET `/usuarios`
- GET `/usuarios/:id`
- POST `/usuarios`
- PUT `/usuarios/:id`
- DELETE `/usuarios/:id`

## Ejecutar localmente

```bash
npm install
npm start
```
